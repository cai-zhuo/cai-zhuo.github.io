Javascript 技巧收集

2020-04-04 1 min read

  1. js通过valueOf获取引用类型的数字，toString获取引用类型的字符串

  2. Math.floor (4.9) === 4

    ~~ 4.9 === 4

    ~~ - 4.9 === -5

  3. 取整 | 0

    1.3 | 0 === 1

    -1.3 | 0 === -1

  4. 判断奇偶数

    const num = 3

    !! (num & 1) === true

    !! (num % 2) === true

  5. 强制参数（mandatory）

    const mandatory = ()=>{
    throw new Error('Missing Parameter!')
}

const foo = (bar = mandatory())=>{
    return bar;
}

  6. 惰性载入函数

    通过覆写函数，使得下一次不用再进行判断

    function foo(){
    if(a != b){
        console.log('a')
    }else{
        console.log('b')
    }
}
//优化后
function foo(){
    if(a != b){
        foo = function(){
            console.log('a')
        }
    }else{
        foo = function(){
            console.log('b')
        }
    }
    foo();	// or return foo()
}

  7. 一次性函数

    let foo = ()=>{
    console.log('1')
    foo = ()=>{
        console.log('2')
    }
}

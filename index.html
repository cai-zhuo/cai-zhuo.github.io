Project 01

Materials AI

A canvas AI for shop & e-commerce posters 面向店铺与电商海报的画布 AI

Materials AI is a canvas-based AI design system. Its main target is shop posters and e-commerce posters — but it can take on more than that. Graphic diagrams and game 2D materials are still in the task list, in progress. So depending on what you need, you can use the web app, the OpenClaw skill, or the npm package below to build your canvas result. Materials AI 是一个基于画布的 AI 设计系统。主要目标是店铺海报和电商海报 —— 但它能做的不止这些。 图形示意图 和 游戏 2D 素材 还在任务列表里，正在推进。 所以根据你的需求，你可以选择下面的网页应用、OpenClaw skill、或者 npm 包来构建你的画布结果。

Project 02

RPG World

A Vampire Survivors-like roguelike 一个类吸血鬼幸存者的 Roguelike

Started as a copy of Vampire Survivors — I want to build my own game on top of this idea. It's not done yet, but the core systems are in place: 94 weapons, 23 characters, 1000+ monsters, weapon evolution, level-up flow, shop, achievements, mobile touch controls. The main remaining work is designing the game materials — sprites, art, and assets that turn the systems into a game with its own identity. Testing is the other piece. 从复刻吸血鬼幸存者开始 —— 我想在这个基础上做出自己的游戏。 还没做完，但核心系统已就位：94 种武器、23 个角色、1000+ 怪物、武器进化、 升级流程、商店、成就、移动端触控。剩下的主要工作是 设计游戏素材 —— 精灵图、美术、资源， 让这些系统真正变成一款有自己风格的游戏。另一部分是测试。

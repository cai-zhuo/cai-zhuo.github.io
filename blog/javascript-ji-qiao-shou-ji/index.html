“数字一大就GG了，性能和Math.floor()不相上下”
2020/8/21
JS中慎用位运算，JS中的位运算继承自JAVA，采取的是用 int 类型进行位运算，但是JS中只有 double 类型，需要从double转换成int，所以会很耗时，性能就显得不那么出色，并且数字变大以后会出现错误。
但是在node中，如果涉及大量的位运算，可以采取编写C++来嵌入进node中，以此来提高性能。

  1. js通过valueOf获取引用类型的数字，toString获取引用类型的字符串

  2. Math.floor (4.9) === 4

    ~~ 4.9 === 4

    ~~ - 4.9 === -5

  3. 取整 | 0

    1.3 | 0 === 1

    -1.3 | 0 === -1

  4. 判断奇偶数

    const num = 3

    !! (num & 1) === true

    !! (num % 2) === true

  5. 强制参数（mandatory）

    const mandatory = ()=>{
    throw new Error('Missing Parameter!')
}

const foo = (bar = mandatory())=>{
    return bar;
}

  6. 惰性载入函数

    通过覆写函数，使得下一次不用再进行判断

    function foo(){
    if(a != b){
        console.log('a')
    }else{
        console.log('b')
    }
}
//优化后
function foo(){
    if(a != b){
        foo = function(){
            console.log('a')
        }
    }else{
        foo = function(){
            console.log('b')
        }
    }
    foo();	// or return foo()
}

  7. 一次性函数

    let foo = ()=>{
    console.log('1')
    foo = ()=>{
        console.log('2')
    }
}

  8. 按位取反巧用

    ~ 0  === -1  //true
~-1 ===   0 //true

const arr = [1,2,3];
